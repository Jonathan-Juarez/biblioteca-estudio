# Biblioteca de Estudio — Landing Page

## Problema original
Landing page de app móvil "Biblioteca de Estudio" — minimalista, tema oscuro, acento azul marino, tipografía Poppins, mockup CSS, totalmente responsiva.

## Decisiones del usuario
- Acento: azul marino (#2F6BFF) + blanco
- Tipografía: Poppins
- Mockup: CSS puro
- Enlaces de descarga: placeholders (#)

## Implementado (2026-05-20)
- Navbar con logo + CTA "Descargar"
- Hero con título, subtítulo, badges, 2 CTAs (App Store / Google Play), stats sociales y phone mockup CSS
- Sección Features: grid de 4 tarjetas (Organización, Notas sincronizadas, Búsqueda, Offline)
- CTA final con bloque destacado y botones repetidos
- Footer con copyright + privacidad + términos
- Responsive móvil ↔ desktop
- Single-file `App.js` con Tailwind
